UNIX family

Most popular modern UNIX systems

  • Ubuntu (Linux)
  • Fedora (Linux)
  • MacOS X (Darwin <- FreeBSD)

Standard: POSIX 1003.1-2008 TC1

Use VirtualBox to try, or install on host (dangerous, but cool).

Ubuntu demo

Terminal

Shell

  • One uses a terminal emulator to access the shell
  • Grandparent of all UNIX shells: Almquist shell (ash)
  • Most popular and widespread shell: bash (Ubuntu, MacOS X)
  • Hipster shells: zsh, fish
  • POSIX standard

All scripts must carry a shebang, e.g.

			#!/bin/sh -e

You can do everything with a shell

Some usefull commands:
  • ls
  • cd
  • pwd
  • kill
  • man
  • which
  • grep/egrep/fgrep

Installing software

Package Managers:
  • Homebrew on MacOSX
  • Chocolatey on Windows
  • apt-get on Ubuntu, DNF on Fedora, etc.

Example:

			apt-get install chromium-browser
			brew install git

Standard streams

  • stdout is for ordinary output (e.g., logs)
  • stderr is for errors output
  • stdin is for reading input
  • More information on standard streams

Console editors

  • VIM
  • Emacs
  • nano

Bad version control

Use VCS

  • VCS - Version Control System
  • Concurrent Versions System (CVS)
  • Subversion (SVN)
  • Mercurial
  • GIT
  • Commits
  • Branches
  • Merges
  • History (log)
  • Distributed operation
    • Pool and push
    • Remotes
